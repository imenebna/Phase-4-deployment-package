In this phase, I deployed my fine tuned BART summarization model through a Streamlit application. The project code and deployment files are stored in GitHub, the large model files are hosted in a Hugging Face model repository, and the live Streamlit application is deployed on Hugging Face Spaces. This structure was necessary because the model file exceeds GitHub's standard upload limit. The final system allows a user to input a long text and generate an abstractive summary through the deployed interface.

GitHub repository: https://github.com/imenebna/Phase-4-deployment-package
Model repository: https://huggingface.co/Imenebna/bart-phase4-model
Live application: https://huggingface.co/spaces/Imenebna/bart-phase4-app
